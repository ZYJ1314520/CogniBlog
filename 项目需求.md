# 智能个人博客系统（AI 增强版）项目需求

## 一、项目定位

前后端分离 + 现代技术栈 + AI 赋能的轻量化个人博客系统，满足文章管理、分类标签、评论互动核心业务，集成 AI 智能助手提升功能亮点，技术栈主流、表结构完整、业务闭环，适合简历展示、面试讲解、快速落地实现。

---

## 二、核心目标

1. 实现完整博客业务，表结构≥5 张，关联关系清晰（一对多、多对多）
2. 采用 Java21 + SpringBoot + Spring AI 最新技术栈，贴合市场需求
3. 集成 Redis 实现缓存优化，体现性能意识
4. 接入 AI 能力，非硬凑功能，贴合博客场景
5. 前后端分离，接口规范、代码简洁、易部署、易演示

---

## 四、功能模块（6 大模块，6 张表）

### 1. 用户模块
- **功能**：注册、登录、用户信息查询 / 修改
- **核心表**：`user`（用户表）

### 2. 文章模块
- **功能**：发布文章、编辑、删除、详情查询、分页列表
- **关联**：所属用户、所属分类、多标签
- **核心表**：`article`（文章表）

### 3. 分类模块
- **功能**：分类新增、编辑、删除、分类下文章查询
- **关联**：一个分类对应多篇文章（一对多）
- **核心表**：`category`（分类表）

### 4. 标签模块
- **功能**：标签管理、文章绑定标签
- **关联**：文章与标签多对多
- **核心表**：`tag`（标签表）、`article_tag`（关联表）

### 5. 评论模块
- **功能**：发布评论、删除、文章评论列表、支持楼中楼（父评论）
- **关联**：用户 - 评论、文章 - 评论
- **核心表**：`comment`（评论表）

### 6. AI 助手模块（亮点加分）
- AI 文章摘要生成
- AI 自动生成标签
- AI 内容润色 / 优化建议
- AI 问答助手（博客使用帮助）

| update_time | datetime | 更新时间 |

| 字段 | 类型 | 说明 |
|------|------|------|

| create_time | datetime | 创建时间 |
| update_time | datetime | 更新时间 |

### 4. tag（标签表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| name | varchar | 标签名称 |
| create_time | datetime | 创建时间 |

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |

## 六、核心业务流程

1. 用户注册 / 登录 → 管理个人信息
2. 创建分类 → 发布文章 → 绑定标签
3. 游客 / 用户查看文章列表、详情、评论
4. 发布评论、互动
5. 使用 AI 助手：生成摘要、自动打标签、润色内容

---

## 七、接口规范

- 统一返回格式：`code` + `msg` + `data`
- RESTful 风格
- 分页统一：`current` + `size`
- 全局异常处理
- 参数校验
