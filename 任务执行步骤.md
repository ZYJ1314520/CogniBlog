# 智能个人博客系统 - 任务执行步骤

## 第一阶段：项目初始化

### Step 1: 创建 Spring Boot 项目结构 ✅ 已完成
- 创建后端 Maven 项目，配置 pom.xml
- 引入依赖：SpringBoot、Spring AI、MyBatis-Plus、MySQL、Redis、Lombok、Hutool
- 配置 application.yml（数据库、Redis、AI 模型）
- 创建启动类
- 配置类：MybatisPlusConfig、RedisConfig、WebConfig、GlobalExceptionHandler
- 通用类：Result、BusinessException

### Step 2: 前端项目初始化 ✅ 已完成
- 使用 Vite + Vue3 创建前端项目
- 安装 Element Plus、Axios、Vue Router、Element Plus Icons
- 配置代理（解决前后端跨域）
- 路由配置：Home、Login、Register、ArticleDetail、Publish、User
- Axios 封装：统一响应处理、Token 拦截、异常处理
- 布局组件：Header、Sidebar

---

## 第二阶段：数据库与实体层

### Step 3: 创建数据库与表结构 ✅ 已完成
- 创建 MySQL 数据库 `cogniblog`
- 执行建表 SQL（6 张表）
- 验证表关联关系
- SQL 脚本位置：`sql/init.sql`
- 初始数据：3 个分类、6 个标签、2 个用户、3 篇文章、3 条评论

### Step 4: 生成实体类（Entity）✅ 已完成
- 创建 User、Category、Article、Tag、ArticleTag、Comment 实体
- 添加字段、注解、关联关系
- 使用 MyBatis-Plus 注解（@TableName、@TableId、@TableField）
- Article、Comment 包含关联对象（user、category、tags、children）

---

## 第三阶段：持久层（MyBatis-Plus）

### Step 5: 编写 Mapper 层 ✅ 已完成
- 创建 UserMapper、CategoryMapper、ArticleMapper、TagMapper、ArticleTagMapper、CommentMapper
- 继承 BaseMapper
- 编写复杂查询（如分页、关联查询）
- ArticleMapper：分页查询、按用户查询
- CommentMapper：按文章查询评论列表

### Step 6: 配置逻辑删除与自动填充 ✅ 已完成
- 配置 create_time、update_time 自动填充（MybatisPlusConfig 中已配置）
- 配置逻辑删除字段（application.yaml 中已配置）

---

## 第四阶段：核心业务层（Service + Controller）

### Step 7: 用户模块 ✅ 已完成
- 实现注册、登录接口（MD5 密码加密）
- 实现用户信息查询 / 修改
- 添加 @Valid 参数校验
- DTO：LoginRequest、RegisterRequest、UserUpdateRequest
- Service：UserService（含 Token 管理）
- Controller：/user/register、/user/login、/user/info、/user/update

### Step 8: 分类模块 ✅ 已完成
- 实现分类 CRUD
- 实现分类下文章查询
- Service：CategoryService（含缓存注解）
- Controller：/category/list、/category/{id}、/category/add、/category/update、/category/delete/{id}、/category/articles/{id}
- 启用 @EnableCaching 缓存支持

### Step 9: 文章模块 ✅ 已完成
- 实现文章发布、编辑、删除
- 实现文章分页列表（关联查询用户、分类、标签）
- 实现文章详情（含浏览量 +1，Redis 计数）
- DTO：ArticleRequest
- Service：ArticleService（含缓存、Redis 浏览量统计）
- Controller：/article/list、/article/my、/article/{id}、/article/add、/article/update、/article/delete/{id}

### Step 10: 标签模块 ✅ 已完成
- 实现标签 CRUD
- 实现文章绑定标签（批量操作）
- Service：TagService（含缓存）
- Controller：/tag/list、/tag/{id}、/tag/add、/tag/update、/tag/delete/{id}、/tag/bind

### Step 11: 评论模块 ✅ 已完成
- 实现评论发布、删除
- 实现评论列表（支持楼中楼展示）
- 关联查询用户信息
- DTO：CommentRequest
- Service：CommentService（含楼中楼树形结构构建）
- Controller：/comment/list、/comment/add、/comment/delete/{id}

---
